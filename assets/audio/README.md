# Ambient Sounds for Focus Timer

Add the following audio files to this folder to enable ambient sounds in the Focus Timer:

| File Name    | Description                    |
| ------------ | ------------------------------ |
| `rain.mp3`   | Rain/rainfall ambient sound    |
| `forest.mp3` | Forest/nature ambient sound    |
| `cafe.mp3`   | Coffee shop/caf√© ambient sound |
| `ocean.mp3`  | Ocean waves ambient sound      |
| `fire.mp3`   | Fireplace/crackling fire sound |

## Requirements

- Format: MP3 or WAV (MP3 recommended for smaller file size)
- Duration: At least 1-2 minutes (will loop automatically)
- Quality: 128kbps or higher recommended

## Free Sources

You can find free ambient sounds at:

- [Pixabay](https://pixabay.com/sound-effects/) - Free for commercial use
- [Freesound](https://freesound.org/) - Various licenses, check before using
- [Zapsplat](https://www.zapsplat.com/) - Free with attribution

## Note

After adding audio files, run `flutter pub get` to update assets.
